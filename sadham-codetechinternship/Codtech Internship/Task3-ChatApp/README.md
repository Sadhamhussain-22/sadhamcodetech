# CODTECH Internship — Task 3 (Java Socket Chat App)

A multithreaded client-server chat system using:
- `ServerSocket` to accept connections
- `Threads` to handle multiple clients
- `Broadcasting` messages to all users in real time

## How to Run
1. Compile:
   javac src\ChatServer.java src\ChatClient.java
2. Start server:
   java -cp src ChatServer
3. Start clients in new terminals:
   java -cp src ChatClient

## Features
✔ Multi-client support  
✔ Username join/leave alerts  
✔ Real-time message broadcasting  
✔ Fully commented code
